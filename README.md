## 🎯 Project Overview

The goal was to consolidate scattered HR data and present a clear 
story about branch performance and salary trends across three 
regions — Alexandria, Cairo, and Tanta. Core question: 
*Where is our investment in people showing the best results?*

---

## 🔧 Step-by-Step Transformation

1. **Consolidation:** Combined multiple tabs into one Master Table 
using Power Query for a single source of truth.
2. **Data Integrity:** Fixed formatting for dates and percentages, 
and used Fill Down logic to resolve missing department names.
3. **Data Cleaning:** Resolved naming inconsistencies in city 
locations to avoid duplicate chart categories.
4. **KPI Design:** Built custom logic to calculate Age, Tenure, 
Performance Benchmarking and Year-over-Year Growth Delta.

---

## 💡 What the Data Told Me

- **Alexandria leads:** Highest salary spend (47%) with the 
strongest performance growth (16%).
- **Education matters:** Bachelor's degree holders consistently 
drive the highest performance scores across all branches.
- **Cairo opportunity:** Higher salaries haven't translated to 
equivalent growth — worth investigating.
